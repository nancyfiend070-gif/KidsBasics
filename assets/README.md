# Assets Folder Structure

This folder contains all images and audio files for the Kids Learning App.

## Folder Structure

```
assets/
├── images/
│   ├── alphabet/     # A.png, B.png, ... Z.png
│   ├── numbers/      # 1.png, 2.png, ... 20.png
│   ├── shapes/       # circle.png, square.png, etc.
│   ├── colors/       # red.png, blue.png, etc.
│   ├── animals/      # lion.png, elephant.png, etc.
│   ├── fruits/       # apple.png, banana.png, etc.
│   ├── body/         # head.png, hand.png, etc.
│   └── quiz/         # Quiz question images
└── audio/
    ├── alphabet/     # A.mp3, B.mp3, ... Z.mp3
    ├── numbers/      # 1.mp3, 2.mp3, ... 20.mp3
    ├── shapes/       # circle.mp3, square.mp3, etc.
    ├── colors/       # red.mp3, blue.mp3, etc.
    ├── animals/      # lion.mp3, elephant.mp3, etc.
    ├── fruits/       # apple.mp3, banana.mp3, etc.
    └── body/         # head.mp3, hand.mp3, etc.
```

## Where to Get Assets

### Free Resources:
1. **Flaticon** (https://www.flaticon.com) - Free icons with attribution
2. **Unsplash** (https://unsplash.com) - Free photos
3. **Pexels** (https://www.pexels.com) - Free stock photos
4. **Pixabay** (https://pixabay.com) - Free images and audio
5. **Freesound** (https://freesound.org) - Free sound effects
6. **Google Fonts** - For text assets

### Paid Resources:
1. **Shutterstock** - High-quality images
2. **Adobe Stock** - Professional assets
3. **Envato Elements** - Subscription-based assets

### Audio Recording:
- Use text-to-speech services (Google TTS, Amazon Polly)
- Record your own voice
- Hire voice actors for professional quality

## Current Status

✅ **App works without assets** - Uses emojis as placeholders
📝 **Add your assets** - Replace emojis with actual images/audio
🎨 **Recommended size**: Images 512x512px or higher
🔊 **Audio format**: MP3, 44.1kHz recommended

## Note

The app currently works with emoji placeholders. When you add actual image/audio files, they will automatically be used instead of emojis.
